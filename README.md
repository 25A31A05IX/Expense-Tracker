Student Details Management System in C++

📌 Introduction

This project is a simple Student Details Management System developed using C++. It collects and displays student information.

🎯 Objective

- To store student details
- To display student information
- To understand input/output in C++

 Technologies Used

- C++
- g++ compiler

 How It Works

The program asks the user to enter student details such as name, roll number, branch, section, email, parent name, contact number, address, and whether the student is a day scholar or hosteller. Then it displays all the entered details.

1. Compile:
   g++ student.cpp -o student
2. Run:
   ./student

✅ Output

The program displays all entered student details.

📈 Result

The program successfully collects and displays student information.
